Pour pouvoir afficher les différentes pages du site en local, il va falloir pour cela deux conditions : 
- Que le fichier soit dézippé dans un dossier projet-L3AN1.
- Modifier le chemin qui se trouve sur activite/script.js dans les url correspondant au graphique et le changer par le chemin de votre réseau local.

Exemple : url : 'http://192.168.64.2/projet-L3AN1/Projet_L3AN1/activite/activite_perso.php'
